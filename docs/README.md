# ModelTrust

**Evaluate which AI models can be trusted in real decision-making.** ModelTrust compares responses across models, measures reliability, detects disagreement, and helps teams decide when human review is needed.

**Tech:** Next.js 16 (App Router), TypeScript (strict), Tailwind CSS, shadcn/ui, Prisma + MySQL, custom MySQL-backed job queue, SSE for live progress.
